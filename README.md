## managment system 前端 叶得力

| Vue | JavaScript | HTML | CSS |



## Demo

| Dashboard | User Profile | Tables  | Notification |


## 开始


### install dependencies
`npm install`
### serve with hot reload at localhost:8080
`npm run dev`
### build for production with minification
`npm run build`
